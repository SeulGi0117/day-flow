# DayFlow - 데모 목업

**DayFlow**는 AI 기반 개인화 루틴 생성 및 목표 관리 앱의 UI/UX 목업입니다.

이 프로젝트는 제품 기획 및 디자인 컨셉을 시각화하기 위한 정적 HTML 데모이며, 실제 백엔드 로직이나 AI 기능은 구현되어 있지 않습니다. 화면 간 연결과 기본적인 인터랙션만 포함되어 있습니다.

## 주요 화면
- 대화형 온보딩 (`onboarding-chat.html`)
- 대시보드 (`dashboard.html`)
- 목표 상세 (`goal-detail.html`)
- 일기 기록 (`diary.html`)
- 회고 리포트 (`report.html`)
- 그룹 챌린지 (`challenges.html`)
- 리커버리 루프 (`recovery-loop.html`)
